# personal-master
基于vue3+Element Plus+Vite2 的个人资源分享网站 代码完全开源



# 使用vite2搭建vue3项目
搭建环境需要使用node，npm
Node.js官网： http://nodejs.cn/
Vite官网：https://vitejs.cn/
## 创建项目
```
npm init vite personal-vue3
```
